# HQSOFT LDP SaaS — Landing Page

Landing Page B2B Enterprise cho HQSOFT, giới thiệu hệ sinh thái giải pháp quản trị phân phối & bán lẻ (DMS, SFA, nRetail, v.v.). Tập trung vào trải nghiệm thị giác ấn tượng, hiệu ứng mượt mà và tương tác động để tăng tỷ lệ chuyển đổi (Lead Generation).

Xây dựng bằng [Next.js](https://nextjs.org) (App Router, React 19, TypeScript, Turbopack).

## Bắt đầu

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000) để xem kết quả.
